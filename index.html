Variables y tipos

Las variables sirven para guardar datos. En JavaScript podemos guardar textos, números, booleanos, arrays u objetos.

let nombre = "Ana";
let edad = 18;
let aprobado = true;

Este ejemplo guarda un texto, un número y un valor verdadero o falso.

Operadores

Los operadores permiten hacer cálculos o comparar valores.

let total = 10 + 5;
let esMayor = total > 12;

Primero se suma 10 + 5 y después se comprueba si el resultado es mayor que 12.

Condicionales

Los condicionales permiten ejecutar un código u otro según una condición.

if (edad >= 18) {
    console.log("Mayor de edad");
} else {
    console.log("Menor de edad");
}

El código comprueba si la edad es mayor o igual que 18.

Bucles

Los bucles repiten instrucciones varias veces sin tener que escribir el mismo código muchas veces.

for (let i = 1; i <= 3; i++) {
    console.log(i);
}

Este bucle muestra los números del 1 al 3.

Arrays

Un array es una lista de datos guardados en una sola variable.

let frutas = ["manzana", "pera", "plátano"];
frutas.push("uva");

El ejemplo crea una lista de frutas y añade una fruta nueva al final.

    Funciones

    Las funciones agrupan instrucciones para poder reutilizarlas cuando las necesitemos.

    function saludar(nombre) {
    return "Hola, " + nombre;
}

    Esta función recibe un nombre y devuelve un saludo personalizado.

    DOM

    El DOM permite acceder y cambiar elementos de una página web usando JavaScript.

    const titulo = document.querySelector("h1");
titulo.textContent = "Nuevo título";

    El código selecciona el primer h1 y cambia su texto.

    Eventos

    Los eventos permiten reaccionar a acciones del usuario, como hacer clic en un botón.

    boton.addEventListener("click", function () {
    alert("Has hecho clic");
});

    El ejemplo ejecuta una acción cuando el usuario pulsa un botón.

    Math

    Math es un objeto de JavaScript que permite hacer operaciones matemáticas útiles.

    let numero = Math.round(4.6);
let aleatorio = Math.floor(Math.random() * 10);

    El primer ejemplo redondea un número y el segundo genera un número aleatorio.

    Strings

    Los strings son cadenas de texto. Se pueden modificar, medir o buscar partes dentro de ellos.

    let texto = "JavaScript";
console.log(texto.length);
console.log(texto.toUpperCase());

    El código muestra la longitud del texto y lo convierte a mayúsculas.

    Fechas

    El objeto Date sirve para trabajar con fechas y horas.

    let hoy = new Date();
console.log(hoy.getFullYear());

    Este ejemplo crea la fecha actual y obtiene el año.

    Formularios

    JavaScript permite comprobar datos de formularios antes de enviarlos.

    formulario.addEventListener("submit", function (evento) {
    evento.preventDefault();
});

    El código evita que el formulario se envíe automáticamente para poder validarlo antes.

    Errores típicos

    Algunos errores comunes son escribir mal nombres de variables, olvidar paréntesis o usar mal los selectores del DOM.

    let nombre = "Ana";
console.log(nom); // Error: la variable correcta es nombre

    El error ocurre porque se intenta usar una variable llamada nom, pero la variable creada se llama nombre.
